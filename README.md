# RustUnZip

RustUnZip is a simple command-line utility written in Rust to extract files from ZIP archives.

## Features

- Extracts all files and directories from a ZIP archive.
- Recursively creates directories as needed during extraction.
- Prints comments associated with files in the ZIP archive (if any).
- Preserves Unix file permissions when extracting on a Unix system.

## Usage

To use RustUnZip, simply provide the path to a ZIP archive as a command-line argument:
 
 
 RustUnZip <path_to_zip_file>
